This project evaluates a pre-campaign bank marketing model that predicts term deposit subscription. 
The dataset comes from the UCI Bank Marketing dataset and is accessed through the Fairlearn library. 
The dataset contains campaign information from a Portuguese bank. 

The project compares and evaluates Logistic Regression, Random Forest, as well as SMOTE for class imbalance. 
From a responsible AI prespective, the project evaluates fairness across age-groups and SHAP explainability.
